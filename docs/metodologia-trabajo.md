# Metodología de trabajo

## Objetivo

Desarrollar el TPV en React manteniendo la apariencia del diseño original y cumpliendo los requisitos funcionales mínimos del trabajo.

## Fases de trabajo

### 1. Preparación del proyecto

- creación del proyecto independiente en `htdocs`
- inicialización de Git
- futura sincronización con GitHub
- creación de documentación base

### 2. Integración visual

- conversión del HTML original a JSX
- reutilización del CSS entregado
- incorporación de assets
- ajuste fino para mantener la misma apariencia

### 3. Separación en componentes

- panel de productos
- tarjeta de producto
- panel de carrito
- línea de carrito
- componente principal coordinador

### 4. Lógica funcional

- gestión del input de peso o cantidad
- alta de productos en carrito
- cálculo de subtotales
- aplicación de cupones
- cálculo del total final
- acción de cobro y reseteo

### 5. Revisión y entrega

- validación de funcionamiento
- revisión visual respecto a la maqueta
- limpieza de código
- documentación del avance por versiones

## Criterios de commits

- un commit por avance lógico
- mensajes claros
- reflejo de la evolución real del proyecto
