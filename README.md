# TpvReactGarazi

Proyecto independiente para desarrollar un TPV funcional con React manteniendo la apariencia del HTML y CSS entregados.

## Requisitos del trabajo

- uso de React
- componentes funcionales
- manejo de estado con `useState`
- misma apariencia base del documento HTML proporcionado
- reutilización de CSS y estructura visual
- listado de productos seleccionados en la parte derecha
- cálculo de subtotal, descuento y total
- aplicación de cupones `SAVE10`, `SAVE20` y `VIP`
- botón de cobro con reinicio del TPV

## Base técnica

- React
- Vite
- Git
- GitHub para seguimiento de versiones

## Estructura principal

- `src/`: aplicación React
- `src/assets/`: imágenes y recursos visuales
- `src/components/`: componentes de interfaz
- `src/data/`: datos base de productos
- `docs/`: metodología, seguimiento y comprobación final

## Puesta en marcha

```bash
npm install
npm run dev
```

## Validación final

La aplicación compila correctamente con:

```bash
npm run build
```
